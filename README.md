# Проект: Mesto

**О проекте**

Одностроничный сайт, который содержит в себе информацию о профиле пользователя и галерею с фотографиями.

**Стек технологий**
* Webpack
* БЭМ
* JS ES6
* CSS 3
* HTML 5
* API
* Promise
* Адаптивная верстка
* ООП
